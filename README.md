# BiasLens

## Agentic AI-Powered Fairness Analysis Platform

BiasLens 2.0 is an advanced fairness analysis tool that uses a multi-agent AI system to automatically detect, analyze, and explain bias in machine learning models.

### Features

-  **5 Specialized AI Agents** for different analysis stages
-  **Automated Data Profiling & Cleaning**
-  **Comprehensive Fairness Metrics**
-  **Interactive Visualizations** 
-  **Natural Language Q&A**
-  **Multi-level Reports** (Technical, Business, Simple)


### Agent Architecture

1. **Data Profiler Agent** - Understands your dataset
2. **Data Cleaning Agent** - Prepares data for analysis  
3. **Bias Detection Agent** - Runs models & computes fairness
4. **Report Generation Agent** - Creates insights & explanations
5. **Conversational Agent** - Answers questions naturally
